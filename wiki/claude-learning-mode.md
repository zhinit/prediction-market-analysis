# Claude Learning Mode

Claude Learning Mode is a built-in Socratic tutoring feature that asks guiding questions instead of providing direct answers. Launched April 2, 2025 as part of Claude for Education (source: anthropic-claude-for-education.md).

## How It Works

Learning Mode uses Socratic questioning to guide learners toward insights. Instead of answering "What's the derivative of sin(x)cos(x)?", Claude asks "What rule would you apply when differentiating a product of two functions?" (source: northeastern-claude-learning-mode.md).

Response patterns:
- Guided discovery (questions before answers)
- Scaffolding for complex problems (breaking into steps)
- Connections to previous learning
- Metacognitive reflection prompts ("Why did you choose that approach?")

## Project Types

Learning Mode is accessed through Claude Projects. Three pre-built templates (source: northeastern-claude-learning-mode.md):

- **Study projects** — master a subject through guided learning
- **Career projects** — explore career development pathways
- **Research projects** — analyze and organize research materials

### Setup

1. Click Projects in the left menu
2. Select Study Project (or Career/Research)
3. Enter a project title
4. Click Create Project

## Learning Mode vs Chat vs Projects

| Feature | Chat | Projects | Learning Mode |
|:--------|:-----|:---------|:--------------|
| Persistent context | No | Yes | Yes |
| Custom system prompts | No | Yes (manual) | Yes (pre-built) |
| Socratic questioning | No | Only if prompted | Built-in |
| Document uploads | Yes | Yes | Yes |

Learning Mode is an extension of Projects with pre-designed system prompts optimized for educational best practices (source: northeastern-claude-learning-mode.md).

## Retention Evidence

"Information discovered through guided questioning is retained 3-4 times longer than information received passively" (cited: Educational Psychology Review, 2024) (source: northeastern-claude-learning-mode.md).

Students using Learning Mode show improved problem-solving confidence, better exam performance, and stronger conceptual understanding (source: northeastern-claude-learning-mode.md).
